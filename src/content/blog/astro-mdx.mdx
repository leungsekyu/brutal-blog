## 2. 纯 CSS 图标

在 MDX 中允许使用 `div` 标签，并且 UnoCSS 和它的预设图标仍能正常使用！

<div class="i-logos-mdx w-26 h-26 -mt-7 -mb-6"></div>

```html
<div class="i-logos-mdx w-26 h-26 -mt-7 -mb-6"></div>
```

```ts
import { defineConfig, presetWind, presetTypography, presetIcons } from 'unocss'

export default defineConfig({
  presets: [
    presetWind(),
    presetTypography(),
    presetIcons({
      collections: {
        logos: async () => {
          const icons = await import('@iconify-json/logos/icons.json')
          return icons.default
        },
      },
    }),
  ],
})
```

## 3. Mermaid 图表（Markdown 中也能使用）

已知问题：使用 `remark-mermaidjs` 插件渲染时，中文和 Emoji 会在部署后出现渲染不全的情况。

> 未知是否与部署环境服务器（本网站部署于 Netlify）有关，在本地使用 remark-mermaidjs 插件渲染时，
> 中文和 Emoji 均能正常显示。

### 对比

#### 中文（使用 remark-mermaidjs 插件渲染）

```mermaid
graph LR
  A[猫 🐱] --> B[老虎 🐯]
```

#### 中文（使用 Mermaid Live Editor 导出 SVG）

![](../../assets/mermaid-cat-tiger.svg)

````mdx
```mermaid
%%{init: {'look': 'handDrawn'}}%%
graph LR
  A[猫 🐱] --> B[老虎 🐯]
```
````

#### 英文（使用 remark-mermaidjs 插件渲染）

```mermaid
graph LR
  A[Cat] --> B[Cat Pro Max]
```

````mdx
```mermaid
%%{init: {'look': 'handDrawn'}}%%
graph LR
  A[Cat] --> B[Cat Pro Max]
```
````

### 踩坑经过

其实 Mermaid 图表最简单的用法是在 Mermaid Live Editor 中编辑，然后在左下方的 Action 折叠栏中
选择下载 SVG 图片或者直接使用生成的 SVG URL
